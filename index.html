Generated bar graphs

Here is the first bar graph that is generated by simply using the following code: 

barGraphIt("data.csv", {"where": "div.simple"});



For a more complex and customized graph that shows all the features of the barGraphIt() function:


dataPath = "data.csv";
vals = {
	"w" : 960, 					// The width of the svg
	"h" : 500, 					// The height of the svg
	"where" : "div .customized",			// Which div element it should be placed in
	"dataValuesLessThanOne" : true,			// Whether the y axis should display values as %
	"margins" : [10,10,20,100],			// top right bottom and left margins
	"yLabel" : "Frequency",				// The label of the y axis
	"tickCount" : 10,				// Approximate number of ticks on the y axis
	"createGrid" : true,				// Whether to show a grid
	"gridColor" : "#FF5722"				// The color of the grid lines
};

barGraphIt(dataPath, vals);



Both graphs are using the default color variables that you can easily change in the bar graph's scss file. They are as follows: 

// Colors
$barcolor 			: #0288D1; 			// bars of the graph
$barHoverColor 			: #FF5722;			// bars when hovered over
$textColor 			: #FFFFFF;			// text of the graph
$axisColor 			: #212121;			// axes' lines
$axisTextColor			: #212121;			// axes' text color

// Axis Labels
$axisFont			: $labelFont;
$axisFontSize			: 1rem;
